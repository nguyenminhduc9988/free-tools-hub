What Is an IP Address?

An IP address (Internet Protocol address) is a unique number assigned to every device connected to the internet. IPv4 addresses look like 192.168.1.1; the newer IPv6 format uses hex digits. Your public IP is assigned by your ISP and can be used to estimate your rough geographic location (typically city-level).

This tool uses the free ipapi.co API for lookups. Note that VPNs and proxies will mask your real IP and location. Use the field above to look up any IPv4 or IPv6 address.
