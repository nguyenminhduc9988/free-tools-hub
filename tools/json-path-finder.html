📖 How to Use

Dot Notation

Navigate nested objects with dots: user.address.city returns "New York". Access arrays with brackets: user.hobbies[0].

JSONPath Syntax

Use $.store.book[0].title to navigate from root. Wildcards: $..author finds all author fields recursively.

Value Search

Type any text in the search box to find all matching values across the entire JSON, regardless of nesting level.

Export Results

Copy matched paths and values to clipboard, or export the full JSON tree as formatted output.
