US & International Postal Codes

A ZIP code (Zone Improvement Plan) is the US postal code system, introduced in 1963. Standard ZIP codes are 5 digits; the extended ZIP+4 adds a hyphen and 4 more digits for finer delivery routing. Other countries use similar systems: Canada (A1A 1A1), UK (e.g. SW1A 1AA), Germany (5-digit), Australia (4-digit).

This lookup uses the free zippopotam.us API which covers ~60 countries. Click any popular ZIP above to see it in action, or enter your own (US uses raw 5-digit; international uses 2-letter country prefix like GB/SW1).
